Seriennummern nachtragen

Dieses Fenster erscheint, wenn mehr als zwei Benutzer angelegt wurden und diese keine gültige Seriennummer haben und keine gültige Lizenzdatei vorhanden ist. Sie müssen eine gültige Lizenzdatei haben oder für jeden Benutzer eine Seriennummer erwerben bevor Sie das Programm weiter verwenden können.

Tun Sie eines der folgenden:

  • Klicken Sie auf 'Lizenzdatei installieren', installieren Sie eine gültige Lizenzdatei und starten Sie das Programm neu
  • Klicken Sie auf 'Neue Seriennummern einfügen' um neue Seriennummern einzufügen, dann auf 'Freie Seriennummern automatisch einfügen'.

Anzahl der freien Seriennummern
Dieses ist die Anzahl von Seriennummern, die für neue Benutzer vergeben werden können.

Anzahl der benötigten Seriennummern
Dieses ist die Anzahl von Benutzern, die keine gültige Seriennummer haben.

Lizenzdatei installieren
Es erscheint der Assistent, mit dem Sie eine gülige Lizenzdatei installieren können.

Neue Seriennummern einfügen
Klicken Sie hier damit das Fenster Seriennummern verwalten erscheint. Hier können Sie neue Seriennummern eingeben.

Freie Seriennummern automatisch verbrauchen
Diese Schaltfläche ist auswählbar, wenn sowohl 'Anzahl der benötigten Seriennummern' als auch 'Anzahl der freien Seriennummern' größer als 0 sind.
Klicken Sie hier damit freie Seriennummern automatisch den Benutzern zugeordnet werden. Jedem Benutzer, der noch keine Seriennummer hat, wird von den freien Seriennummern eine zugewiesen.

Anschließend müssen Sie das Programm neu starten.
