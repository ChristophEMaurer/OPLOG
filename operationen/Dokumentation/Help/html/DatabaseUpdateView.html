Datenbank Update

Wenn dieses Fenster erscheint, haben Sie eine neue Programmversion aus dem Internet heruntergeladen, welche eine Änderung an der Datenbankstruktur vornehmen muss.

Damit diese Änderungen vorgenommen werden können, sollte außer Ihnen kein anderer Benutzer auf die Datenbank zugreifen.
Sie sollten eine Sicherung der Datenbank erstellen, bevor Sie weiter fortfahren.

Wenn verschiedenen Benutzer auf dieselbe Datenbank zugreifen wie Sie, müssen diese ebenfalls deren Version von OP-LOG aktualisieren, da Programm und Datenbank zueinander passen müssen.

Folgen Sie den Anweisungen, die im Detail in diesem Fenster angegeben sind.
