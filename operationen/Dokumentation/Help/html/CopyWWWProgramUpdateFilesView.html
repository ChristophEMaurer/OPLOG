Aktuelle Programmversion aus dem Internet in ein lokales Verzeichnis kopieren

Wenn Sie keine Internetverbindung haben, und Sie das Programm aus einem lokalen Verzeichnis aktualisieren möchten, so müssen Sie die Programmdateien einmal aus dem Internet in dieses Verzeichnis kopieren.

Anschließend können Sie von den Rechnern, die keinen Internet-Zugriff haben, auf dieses Verzeichnis zugreifen.
