Weiterbildungszeiträume

In diesem Fenster können Sie Zeiträume festlegen.

Das Fenster ist etwas kompliziert zu bedienen. Probieren Sie es einfach aus. Wenn Sie einen Zeitpunkt oben anlegen sehen Sie unten die Zeiträume, die Sie angelegt haben. Im Beispiel sind Zeiträume von jeweils einem Jahr angeleggt.

Klicken Sie auf 'Speichern', damit Ihre Einstellungen nicht verloren gehen!

Die Soll-Ist Zahlen werden dann nach diesen Zeiträumen getrennt aufgeführt wenn Sie im Fenster Weiterbildungs-Richtlinien auf 'Drucken in BDC Logbuch Format' oder 'Drucken' klicken.
