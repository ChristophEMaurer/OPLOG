Neue Prozedur manuell anlegen

In diesem Fenster können Sie eine oder mehrere Prozeduren, die Sie ausgeführt haben, manuell anlegen.

Die Prozedur wird für den Arzt angelegt, der angemeldet ist, sie können keine Prozeduren für einen anderen Arzt anlegen.

Prozeduren werden normalerweise nicht manuell eingegeben, da sie durch den Datenimport automatisch in das System gelangen.

Achten Sie darauf, dass alle OPS-Kodes von Prozeduren, die Sie durchgeführt haben, auch einer Richtlinie zugeordnet sind, ansonsten kann die Prozedur später nicht automatisch einer Richtlinie zugeordnet werden.

OPS-Kode/OPS-Text/Anzeigen

Geben Sie den Anfang eines OPS-Kodes in das Feld OPS-Kode ein oder einen Teile einer Bezeichnung in das Feld OPS-Text. Wenn Sie nun auf Anzeigen klicken, werden alle hierzu passenden Prozeduren agezeigt.

Eine ausgeführte Prozedur manuell eingeben

Füllen Sie zunächst die Liste mit Prozeduren, indem Sie in den Feldern OPS-Kode und OPS-Text Werte eingeben und Anzeigen klicken. Klicken Sie nun auf die Prozedur, die Sie eingeben möchten, füllen Sie die Felder Anzahl, Fallzahl usw. aus und klicken Sie auf Prozedur einfügen. Das Feld Anzahl enthält hierbei den Wert 1, wenn Sie eine Prozedur einfügen.
