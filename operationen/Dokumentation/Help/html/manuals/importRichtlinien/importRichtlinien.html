Weiterbildungsrichtlinien importieren

Falls Weiterbildungsrichtlinien fehlen, können Sie diese aus einer Datei importieren. Es werden immer nur Richtlinien importiert, die noch nicht vorhanden sind. Identische Richtlinien, die also schon vorhanden sind, werden nicht doppelt eingefügt. Am besten löschen Sie zunächst alle Richtlinien, bevor Sie die neuen aus einer Datei einlesen.

Lesen Sie Sie hierzu auch:

  • die Hilfedatei:
    Im Internet auf der Seite http://op-log.de/DownloadProgramm.aspx
    Im Programm: 'Hilfe > Hilfedatei (chm)'.

Das Facharztgebiet muss vorhanden sein

Das Facharztgebiet, für welches Sie die Weiterbildungsrichtlinien importieren wollen, muss bereits angelegt sein. Wenn das nicht der Fall ist, legen Sie es an unter 'Offizielle Dokumente > Facharztgebiete': Füllen Sie die Felder 'Gebiet', 'Bemerkung' und 'Herkunft' und klicken Sie dann auf "Einfügen". Erst jetzt können Sie es für den Import auswählen.

Facharztgebiete (Offizielle Dokumente > Facharztgebiete)

Von Allgemeinchirurgie bis Visceralchirurgie:

Dateien von der Internetseite www.op-log.de/DownloadProgramm.aspx herunterladen

Beispielhaft werden hier die Richtlinien für das Facharztgebiet "Allgemeinchirurgie" importiert. Analog gehen Sie für ein anderes Gebiet vor.

1. Gehen Sie auf die Seite http://www.op-log.de/DownloadProgramm.aspx und scrollen Sie nach unten bis Sie zu der Tabelle "Richtlinien" kommen:

2. Bei der gewünschten Datei, z.B. Allgemeinchirurgie (oder auf die von Ihnen gewünschte Datei) rechts auf "Download" mit der rechten Maustaste klicken, damit das Kontextmenü aufgeht.

3. Im Kontextmenü "Save Target As...", auf Deutsch wohl "Ziel speichern unter" klicken.

Es erscheint das "Datei-Speichern" Fenster, hier ein Verzeichnis auswählen, das man später wiederfindet und "Speichern" klicken. In unserem Beispiel wird die Datei "Allgemeinchirurgie.txt" im Verzeichnis "c:\Richtlinien" gespeichert, der vollständige Name ist also "c:\Richtlinien\Allgemeinchirurgie.txt":

Datei aus dem Programm "OP-LOG" importieren

Sie haben nun die Datei "Allgemeinchirurgie.txt", die Weiterbildungsrichtlinien enthält, aus dem Internet heruntergeladen und auf Ihrem PC unter "c:\Richtlinien\Allgemeinchirurgie.txt" gespeichert.

Nun starten Sie das Programm "OP-LOG" und klicken im Menü auf "Extras > Datenimport > Richtlinien importieren Assistent"

Es erscheint der Assistent:

Klicken Sie 'Weiter'.

Wählen Sie 'Allgemeinchirurgie' (bzw. das Gebiet, für das Sie die Zuordnungen importieren möchten), und klicken Sie dann 'Weiter >>'.

Klicken Sie 'Datei auswählen' und wählen Sie die Datei 'c:\Richtlinien\Allgemeinchirurgie.txt' aus, die Sie vorher heruntergeladen haben, so dass diese im Kasten steht wie im Bild zu erkennen.

Klicken Sie 'Weiter'

Kontrollieren Sie, ob die in der Zusammenfassung angegebenen Informationen stimmen und klicken Sie dann 'Fertigstellen'.

Der Import läuft nun durch und am Ende erscheint das Fenster

Klicken Sie Schließen.

Importierte Richtlinien ansehen

Klicken Sie im Programm auf den Menüpunkt Offizielle Dokumente > Weiterbildungsrichtlinien'

Wählen Sie ganz oben aus der Liste 'Allgemeinchirurgie' aus, nun sehen Sie die Weiterbildungsrichtlinien der Allgemeinchirurgie. die Sie gerade importiert haben.
