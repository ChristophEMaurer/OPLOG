Dateien übertragen

Siehe auch

  • Datenexport > OPSKatalog für Handy
  • Datenimport > Operationen vom Handy

Beschreibung

Dieses Fenster finden Sie sowohl unter Datenimport als auch unter Datenexport. Hier können Sie Dateien von und zu einem anderen Handy übertragen. Dieses funktioniert z.Z. mit folgenden Geräten:

  • iPhone/iPod Touch/iPad

Datei senden (Computer -> Handy)

Sie senden den OPSKatalog, den Sie unter Datenexport > OPSKatalog für Handy erstellt haben, an ein Handy. Dort können Sie den OPSKatalog mit 'OPSKatalog aus einer Datei einlesen' in das Programm auf dem Handy einlesen.

  1. Erstellen Sie unter Datenexport > OPSKatalog für Handy einen kleinen OPSKatalog für das Handy.
     
  2. Wählen Sie auf dem Handy 'Daten mit WiFi empfangen'. Das Handy wartet jetzt darauf, dass sich der Computer mit ihm verbindet und eine Datei schickt.
     
  3. Öffnen Sie dieses Fenster. Auf dem Handy werden die IP-Adresse des Handys und der Port angezeigt, über den die Daten verschickt werden. Stellen Sie diese beiden Werte hier ein. Sie können den Port auf dem Handy und hier ändern (beide müssen gleich sein!!!), nicht aber die IP-Adresse, da diese von dem Handy bestimmt wird.
     
  4. Klicken Sie hier 'Datei senden' und wählen Sie den oben erzeugten OPSKatalog aus. Diese Datei wird an das Handy übertragen.
     
  5. Wählen Sie auf dem Handy 'OPSKatalog aus einer Datei einlesen' und wählen Sie die Datei aus, die soeben an das Handy übertragen wurde. Damit haben Sie einen OPSKatalog auf dem Handy zur Verfügung, aus dem Sie Ihre Operationen auswählen können.
     
Datei empfangen (Handy -> Computer)

Sie empfangen die Operationen, die Sie auf dem Handy angelegt und mit 'Meine Operationen in eine Datei schreiben' in eine Datei geschrieben haben. Diese Datei wird vom Handy an den Computer geschickt.

  1. Wählen Sie auf dem Handy 'Meine Operationen in eine Datei schreiben'. Merken Sie sich diesen Dateinamen!!!
     
  2. Öffnen Sie auf dem Computer dieses Fenster. Klicken Sie 'Empfangen' und wählen Sie einen Dateinamen aus, unter dem die Operationen, die das Handy schickt, gespeichert werden sollen. Merken Sie sich diesen Namen! Der Computer wartet jetzt darauf, dass sich das Handy mit ihm verbindet und eine Datei schickt.
     
  3. Wählen Sie auf dem Handy 'Daten mit WiFi senden'. Geben Sie auf dem Handy die IP-Adresse und den Port ein, die in diesem Fenster angezeigt werden. Sie können den Port auf dem Handy und hier ändern (beide müssen gleich sein!!!), nicht aber die IP-Adresse, da diese von diesem Computer bestimmt wird.
     
  4. Wählen Sie auf dem Handy die Datei aus, in die sie die Operationen geschrieben haben, indem Sie auf die entsprechende Textbox auf dem Handy drücken.
     
  5. Wählen Sie auf dem Handy 'Senden'. Die ausgewählte Datei wird an den Computer übertragen.
     
  6. Wählen Sie auf dem Computer 'Datenimport > Operationen vom Handy'. Hier können Sie die Datei, die soeben vom Handy geschickt wurde, einlesen.
     

IP-Adressen und Ports

Eine IP-Adresse ist eine weltweit eindeutige Nummer eines Computers. Ein Port ist eine Nummer, z.B. 8080, und ist eine Art 'Steckdose' innerhalb eines Computers. Ein Computer kann soviele verschiedenen Ports haben, wie es verschiedenen Zahlen gibt, also ziemlich viele. Bei Computern werden bestimmte Daten immer an einen festgelegten Port geschickt, auf diesem Port 'horcht' und empfängt der andere Computer.

Webseiten z.B. werden immer über Port 80 verschickt, sonst 'passen' sie nicht in den Computer. ICQ verschickt seine Daten über Port 5190. Würde man Webseiten an den ICQ-Port 5190 schicken, würde sich ICQ verschlucken. Ebenso macht es keinen Sinn, eine ICQ-Nachricht an den Port 80 zuschicken, da der Computer dort eine Webseite erwartet und mit der ICQ-Nachricht nichts anfangen könnte.

Die IP-Adresse eines Computers entspricht der Post-Adresse eines Hauses. Diese ist auch weltweit eindeutig. Die Post, der Briefträger, die Ämter und viele Andere wissen, wo das Haus mit einer bestimmten Adresse steht. Bei Computern gibt es ebenfalls spezielle Computer, die wiederum wissen, wo ein Computer mit einer bestimmten IP-Adresse steht.

Dem Port entspricht bei einem Haus der 'Steckdose', an die ein spezieller Gegenstand geliefert werden soll. So wie ein Computer viele Ports hat, hat ein Haus viele solcher 'Steckdosen': den Briefkasten, den Wasserhahn, das Häuschen für die Mülltonne. Die Post kommt in die Steckdose 'Briefkasten', ein Gartenschlauch würde an die Steckdose 'Wasserhahn' geliefert, und die Müllabfuhr stellt die Mülltonne in die Steckdose 'Mülltonnenhäuschen'. Man kann die Mülltonne nicht an den Wasserhahn anschließen. Den Brief kann man ins Mülltonnenhäuschen legen, aber das würde nicht funktionieren.
