Akademischer Lebenslauf

Hier geben sie die Stationen Ihres akademischen Lebenslaufes ein.

Wenn Sie einen Zeitraum, etwas für das Studium eingeben möchten, füllen Sie sowohl das Beginndatum als auch das Enddatum aus.

Wenn Sie einen Zeitpunkt eingeben möchten, füllen Sie das Beginndatum aus und lassen das Endedatum leer.
