Benutzer/Abteilungen/Rollen/Rechte - Übersicht

In diesem Fenster erhalten Sie eine Übersicht über einen Benutzer. Man sieht, welchen Abteilungen und Rollen er angehört. Man sieht, welchen Weiterbildern er zugeordnet ist und falls er selber Weiterbilder ist, welche anderen Benutzer ihm zugeordnet sind. Außerdem sieht man, welche Rechte er insgesamt besitzt. Ein Benutzer erhält alle Rechte von allen Rollen, denen er angehört.
