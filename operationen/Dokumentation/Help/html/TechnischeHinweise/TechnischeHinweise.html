Technische Hinweise

OP-LOG ist mit dem Microsoft .NET Framework 2.0 in C# erstellt und unterstützt die Datenbanken Microsoft Access, Microsoft SQL Server und MySQL.

Installation

Beachten Sie die Informationen auf der Web-Site, dort stehen alle notwendigen Dateien zum Download zur Verfügung.

OP-LOG installieren

Installieren Sie das Programm, indem Sie auf die entsprechende Datei im Internet unter http://www.op-log.de auf der Download Seite klicken und folgen Sie den Anweisungen.

Voraussetzungen

  • Windows 2000, Windows XP (Home und Professional), Windows Vista und Windows 7
  • Das .NET Framework 2.0 (Version 2.0.50727) muss installiert sein
    Wenn Sie das Setup starten, erhalten Sie eine Meldung, wenn das .NET Framework 2.0-Framework nicht installiert ist. Sie müssen es dann installieren und das Setup von OP-LOG erneut ausführen.

.NET Framework 2.0 installieren

Downloaden Sie die Datei dotnetfx.exe von der Microsoft Windows Internet Site herunter.

Starten Sie anschließend die herunter geladene Datei dotnetfx.exe und folgen Sie den Anweisungen. Bei Problemen fragen Sie ihren Systemadministrator.
