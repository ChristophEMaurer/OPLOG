Eigene Dateien

Unter diesem Hauptmenüpunkt sehen Sie alle Eigenen Dateien, die Sie in das System kopiert haben. Diese stehen nun jedem Benutzer zur Verfügung.

Sie müssen mit "Verwaltung > Eigene Dateien" Dateien hinzufügen, damit diese in diesem Menü erscheinen. Andernfalls ist dieses Menü leer.

Dieses Menü ist anfangs leer. Solange es leer ist, erscheint der Hinweis 'Keine eigenen Dateien vorhanden'.

Erst nachdem Sie mit "Verwaltung > Eigene Dateien" Dateien hinzugefügt haben, erscheinen diese in diesem Menü.

Unter diesem Menüpunkt erscheinen alle Dateien, die unter 'Verwaltung > Eigene Dateiarten' und 'Verwaltung > Eigene Dateien' in das System kopiert wurden.

Jede Eigene Dateiart erzeugt einen Untermenüpunkt unter dem Hauptmenüpunkt 'Eigene Dateien' und jede Eigene Datei erzeugt in diesem Untermenüpunkt einen Eintrag, der ausgewählt werden kann, wodurch die Datei angezeigt wird.
