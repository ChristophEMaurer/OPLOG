Installieren für mehrere Benutzer auf mehreren PCs

Verzeichnisse als vertrauenswürdig einstellen
Richten Sie das Programmverzeichnis und das Verzeichnis für die gemeinsamen Daten ein wie unter Verzeichnisse als vertrauenswürdig einrichten beschrieben.

Wählen Sie diese Installationsart, wenn

  • Sie das Programm mit mehreren Personen benutzen und auf dieselben Daten zugreifen möchten.
  • Jeder Benutzer das Programm auf seinem PC installiert hat und nur die Daten gemeinsam genutzt werden.

Voraussetzungen für die Installation

  • Sie müssen Administrator sein und Lese- und Schreibrechte in dem Verzeichnis haben, in dem das Programm installiert ist.
  • Sie müssen Administrator sein und Lese- und Schreibrechte in dem Verzeichnis haben, in dem die gemeinsamen Daten liegen.

Voraussetzungen zum Starten des Programmes

  • Sie müssen Leserechte in dem Verzeichnis haben, in dem das Programm installiert ist.
  • Sie müssen Lese- und Schreibrechte in dem Verzeichnis für die gemeinsamen Daten haben.

Hinweis

  • Wenn ein Benutzer eine neue Version installiert, wobei die Datenbankstruktur aktualisiert wird, müssen alle anderen Benutzer ihr Programm auch auf den neuesten Stand bringen.

Installation
Wählen Sie bei der Installationsart '2. Mehrere Benutzer' aus.

Im nächsten Schritt sehen Sie, dass das Programm auf dem PC in C:\Program Files\OP-LOG liegt, die Datenbank aber im Verzeichnis \\cmaurer\LogbuchDaten liegt.
