Rollen und Rechte

OP-LOG enthält eine vollständige Rechteverwaltung. Jedem Menüpunkt und jeder Schaltfläche ist ein Recht zugeordnet. Alle vorhandenen Rechte können Sie unter Zuordnungen von Rechten zu Rollen sehen. Rechte können nicht geändert oder hinzugefügt werden, sie sind fester Bestandteil des Programmes. Wenn durch ein Programm-Update ein neuer Menüpunkt hinzukommt, ist automatisch auch ein neues Recht für diesen Menüpunkt vorhanden.

Im Fenster Zuordnungen von Rechten zu Rollen kann einer Rolle Rechte zugewiesen werden. Ein Recht kann einer oder mehreren Rollen zugeordnet werden.

Im Fenster Zuordnungen von Benutzern zu Rollen wird festgelegt, welche Benutzer zu einer Rolle gehören. Gehört ein Benutzer zu einer Rolle, so erhält er alle Rechte dieser Rolle. Gehört ein Benutzer zu mehreren Rollen, so erhält er alle Rechte von allen Rollen, denen er angehört.

Dadurch, dass einer Rolle bestimmte Rechte zugewiesen werden und Benutzer einer bestimmten Rolle angehören, lassen sich Aufgaben und auch Zugriffe im Programm hervorragend steuern.

Im Fenster Benutzer/Abteilungen/Rollen/Rechte - Übersicht kann man für einen einzelnen Benutzer sehen, welchen Abteilungen und Rollen er angehört, und welche Rechte er insgesamt erhält.

Sichtweite auf die Daten

Ein normaler Benutzer sieht nur seine eigenen Daten.

Ein Benutzer, der Weiterbilder ist, sieht seine eigenen Daten und alle Daten der ihm zugeordneten Benutzer, egal aus welcher Abteilung diese stammen.

Ein Benutzer, der das Recht 'alle Chirurgen aus meinen Abteilungen sehen' besitzt, sieht alle Daten von allen Chirurgen aus allen Abteilungen, denen er selber angehört. Dieses Recht muss einem Benutzer explizit zugeordnet werden.

Ein Benutzer, der sowohl Weiterbilder ist als auch das Recht 'alle Chirurgen aus meinen Abteilungen sehen' besitzt, sieht seinen eigenen Daten, die Daten aller Chirurgen, die ihm zugeordnet sind, sowie alle Daten von allen Chirurgen aus all denen Abteilungen, denen er selber angehört.

Ein Benutzer, der das Recht 'alle Chirurgen sehen' besitzt, sieht alle Daten von allen Chirurgen. Dieses Recht muss einem Benutzer explizit zugeordnet werden.
