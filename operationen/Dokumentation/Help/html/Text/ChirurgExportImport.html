Chirurgendaten exportieren und importieren

Siehe auch Extras > Datenimport > Arzt importieren Assistent.
Siehe auch Extras > Datenexport > Arzt exportieren Assistent.

Sie können exportierte Daten nur innerhalb derselben Programm-Version importieren.

Beispiel
Wenn Sie aus der Version 1.20.1 Chirurgen-Daten exportieren, müssen Sie die erzeugte Datei auch wieder in ein Programm der Version 1.20.1 importieren.

Was mache ich, wenn ich eine alte, exportierte Datei in eine neuere Programmversion importieren möchte?
Angenommen, Sie haben aus der Version 1.20.1 Chirurgen-Daten exportiert, haben inzwischen Version 1.21.0 und möchten jetzt die alte exportierte Datei im Format 1.20.1 in die aktuelle Version 1.21.0 importieren.
Da sie die exportieren Daten im alten Format nicht in eine neuere Programmversion importieren können, müssen Sie folgendes tun.

  1. Installieren Sie alte Version 1.20.1
  2. Importieren Sie die exportierte Datei im alten Format 1.20.1
  3. Updaten Sie auf die neueste Version 1.21.0, indem Sie auf Extras > Auf neue Programmversion überprüfen klicken.
