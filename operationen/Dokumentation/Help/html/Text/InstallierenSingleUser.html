Installieren für einen einzelnen Benutzer

Wählen Sie diese Installationsart, wenn

  • Sie das Programm als einziger verwenden, z.B. zu Hause auf Ihrem PC.

Voraussetzungen für die Installation

  • Sie müssen Administrator sein und Lese- und Schreibrechte in dem Verzeichnis haben, in dem das Programm installiert ist.

Voraussetzungen zum Starten des Programmes

  • Sie müssen Administrator sein und Lese- und Schreibrechte in dem Verzeichnis haben, in dem das Programm installiert ist.

So sichern Sie Ihre Daten
Lesen Sie hier, wie Sie Ihre Daten sichern können, falls Sie Ihren Rechner neu installieren müssen.

Installation
Wählen Sie bei der Installationsart '1. Ein Benutzer' aus.

Im nächsten Schritt sehen Sie, dass sowohl das Programm als auch die Daten auf Ihrem PC in demselben Verzeichnis liegen.
