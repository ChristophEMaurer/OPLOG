Auf neue Programmversion aus einem Verzeichnis überprüfen

Das Programm OP-LOG wird weiterentwickelt. Wenn die Überprüfung einer neuen Version aus dem Internet nicht möglich ist, weil Sie zum Beispiel keine Internetverbindung haben, können Sie das Programm stattdessen aus einem lokalen Verzeichnis aktualisieren. Hierzu muss natürlich eine andere Person bei seinem Update aus dem Internet dort eine Kopie abgelegt haben.

Siehe hierzu 'Extras > Optionen > Programm Update'.
