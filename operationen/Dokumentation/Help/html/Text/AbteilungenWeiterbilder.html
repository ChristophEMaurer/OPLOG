Abteilungen und Weiterbilder

Wenn OP-LOG in einer Klinik mit mehreren Abteilungen genutzt wird, ist es sinnvoll, diese Struktur auch im Programm nachzubilden.

Abteilungen

Abteilungen dienen der Abgrenzung der Daten, die bei Auswertungen berücksichtigt werden.
Die Daten von mehreren Abteilungen können in einer Installation von OP-LOG gehalten werden. Jede Abteilung hat nur Zugriff auf ihre eigenen Daten.

Ein Chirurg kann keiner, einer oder mehreren Abteilungen angehören und sieht immer alle seine eigenen Daten.

Was hat es für Auswirkungen, zu einer Abteilung zu gehören?
Wenn der angemeldete Benutzer das Recht 'alle Chirurgen aller meiner Abteilungen sehen' hat, so kann dieser in Auswertungen auf die Daten aller Chirurgen aus allen Abteilungen zugreifen, denen er angehört.

Weiterbilder

Ein Chirurg ist ein Weiterbilder, wenn dieses im Fenster Operateur bearbeiten eingestellt wurde.
Chirurgen werden einem Weiterbilder im Fenster Zuordnungen von Chirurgen zu Weiterbildern zugeordnet.

Was hat es für Auswirkungen, Weiterbilder zu sein?
Wenn der angemeldete Benutzer Weiterbilder ist, kann dieser in Auswertungen auf die Daten aller Chirurgen zugreifen, die ihm zugeordnet sind. Die ihm zugeordneten Chirurgen können auch aus verschiedenen Abteilungen stammen.
