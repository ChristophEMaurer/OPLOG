Programm installieren

Öffnen Sie einen Web-Browser und navigieren Sie zu der Internet Adresse www.op-log.de. Gehen Sie hier auf die Download-Seite und klicken Sie auf download rechts neben OP-LOG Setup (.exe):

Klicken Sie im folgenden Fenster auf Ausführen/Run, um das komprimierte Setup-Programm herunterzuladen:

Internet ExplorerFirefox

Das komprimierte Setup-Programm wird jetzt heruntergeladen:

Internet ExplorerFirefox

Klicken Sie wieder auf Ausführen/Run, hiermit erteilen Sie die Erlaubnis das Setup auszuführen:

WindowsWindows 7

Klicken Sie jetzt Setup. Es wird das komprimierte Setup-Programm entpackt und gestartet:

Folgen Sie den Anweisungen des Installationsprogramms:
