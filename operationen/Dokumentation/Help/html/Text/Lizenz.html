Seriennummern

Die Software Logbuch-Weiterbilung selber ist immer kostenlos, Sie können die Software unter www.op-log.de herunterladen und installieren. Es steht immer die vollständige Funktionalität der Software zur Verfügung.

Bis zu zwei Ärzte können Sie kostenlos anlegen und mit diesen Daten arbeiten. Möchten Sie mehr als zwei Ärzte anlegen, so benötigen Sie für jeden weiteren Arzt eine Seriennummer.

Seriennummer können Sie auf www.op-log.de bestellen. Sie können auch im Programm selber auf Extras > Seriennummern im Internet bestellen klicken, um schnell zu der Webseite zu gelangen.

Damit Sie mehrere Seriennummern nicht von Hand eingeben müssen, können Sie diese unter Extras > Seriennummern verwalten einfügen.

Unter Extras > Optionen > Seriennummern können Sie einstellen, dass die Seriennummern automatisch verbraucht und freigegeben werden.
