Eine Prozedur mehrfach einfügen

Wenn Sie unter Bearbeiten > Neue Prozedur manuell anlegen eine Prozedur manuell erfassen, können Sie einen OPS-Kode gleich mehrfach anlegen. Zur Sicherheit werden Sie gefragt, ob das auch wirklich Ihre Absicht war.

Achten Sie darauf, dass alle OPS-Kodes von Prozeduren, die Sie durchgeführt haben, auch einer Richtlinie zugeordnet sind, ansonsten kann die Prozedur später nicht automatisch einer Richtlinie zugeordnet werden.
