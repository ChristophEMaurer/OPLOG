Dienststellungen

Hier werden Dienststellungen verwaltet. Alle hier vorhandenen Dienststellungen können beim Bearbeiten eines Arztes ausgewählt werden.

Auch wenn Sie unter 'Bearbeiten > Einschätzungen oder Empfehlungen' einen Eintrag erzeugen, müssen Sie für die Person, über die eine Einschätzung abgegeben wurde, eine Dienststellung auswählen.
