Rollen

In diesem Fenster legen Sie Rollen an.
Ein Benutzer kann keiner, einer oder mehreren Rollen angehören.
Eine Rolle kann Rechte enthalten wie zum Beispiel das Recht, auf einen Menüpunkt klicken zu dürfen. Wenn ein Benutzer einer Rolle angehört, hat er alle Rechte der Rolle.
Wenn ein Benutzer mehreren Rollen angehört, hat er alle Rechte von allen diesen Rollen.
