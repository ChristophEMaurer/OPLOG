Weiterbildungsrichtlinien

In diesem Fenster werden die Richtlinien der Bundesärztekammer verwaltet. Zu jedem Facharztgebiet gehört eine eigene Menge von Richtlinien.

Feld Richtzahl

Bei der manuellen Eingabe beachten Sie folgendes:

  • für eine Überschrift wie '5 - Operative Eingriffe, davon' geben Sie als Richtzahl eine 0 ein
  • um Basiskenntnisse (BK) einzugeben, geben Sie bei Richtzahl 'BK' ein

Neue Richtlinie einfügen

Um eine Richtlinie einzufügen, klicken Sie zunächst im Kasten 'Gebiete' auf das Facharztgebiet, für das Sie eine Richtilinie einfügen möchten. Füllen Sie die Felder Nr., Richtzahl und Untersuchungs- und Behandlungsmethode aus und klicken Sie auf Einfügen.

Richtlinie ändern

Klicken Sie im Kasten "Richtlinien" auf die Richtlinie, die Sie ändern möchten. Ändern Sie die Werte in den Feldern Nr., Richtzahl und Untersuchungs- und Behandlungsmethode und klicken Sie anschließend auf Ändern.

Richtlinie löschen

Klicken Sie im Kasten "Richtlinien" auf die Richtlinie, die Sie löschen möchten. Klicken Sie jetzt auf Löschen.

Alte und neue Richtlinie für ein Facharztgebiet

Ein Facharztgebiet kann auch mehrfach vorhanden sein, zum Beispiel weil man alte und neue Richtlinien gleichzeitig benötigt. Hierzu legt man unter 'Offizielle Dokumente > Facharztgebiete' die benötigten zusätzlichen Facharztgebiete an und kann hier dem Facharztgebiet Richtlinien zuordnen.

Einer Richtlinie kann unter 'Bearbeiten > Zuordnung von Richtlinien zu Prozeduren' eine Menge von OPS-Kodes zugeordnet werden, so dass die vorhandenen Prozeduren eines Chirurgen den Richtlinien automatisch zugeordnet werden können.

Richtlinien exportieren/importieren

Sie können Richtlinien exportieren und importieren. Hierzu starten Sie den jeweiligen Assistenten unter 'Extras > Datenimport/Datenexport'.
Sie importieren Richtlinien, wenn Sie diese nicht selber eingeben möchten und Ihnen jemand die von ihm erstellten Richtlinien als Exportdatei zur Verfügung gestellt hat.
