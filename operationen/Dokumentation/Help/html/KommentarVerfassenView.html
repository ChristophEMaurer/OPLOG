Neue Einschätzung/Empfehlungen anlegen

In diesem Fenster wird eine neue Einschätzung oder Empfehlung angelegt. Nur ein Benutzer, der als Administrator angemeldet ist, kann eine solche Einschätzung über einen anderen Arzt anlegen. Sobald z.B. der Chefarzt eine Einschätzung über einen anderen Arzt in der Ausbildung angelegt hat, kann dieser Arzt in der Ausbildung eine Stellungnahme hinzufügen.

Weder die Einschätzung noch die Stellungnahme kann jemals verändert oder gelöscht werden.
