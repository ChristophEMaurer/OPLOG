Facharzt-Gebiete

Hier können Sie die Facharzt-Gebiete verwalten.

Jedem Facharzt-Gebiet können Sie Richtlinien zuweisen.

Wenn Sie für eine Art von Facharzt alte und neue Richtlinien gleichzeitig verwenden möchten, müssen Sie dieses Facharzt-Gebiet zweimal anlegen und können dann diesen Facharzt-Gebieten verschiedene Richtlinien zuweisen.
