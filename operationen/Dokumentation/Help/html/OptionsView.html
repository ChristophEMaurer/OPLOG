Optionen

Proxy Server

nach oben

Damit die automatische Überprüfung auf eine neuere Version von OP-LOG im Internet durchgeführt werden kann, müssen Sie eventuell hier die Einstellungen für den Proxy Server ändern.

Fragen Sie hierzu an Ihren System-Administrator.

Programm Update

nach oben

Hier stellen Sie ein, ob bei jedem Programmstart überprüft wird, ob eine neue Version im Internet vorhanden ist.

Automatischer Prozeduren-Import

nach oben

Hier stellen Sie ein, ob und wie der automatische Import aller Dateien aus einem bestimmten Verzeichnis durchgeführt wird.

Siehe hierzu Extras > Datenimport > Automatischen Prozeduren-Import durchführen.

Seriennummern

nach oben

Wenn einen Arzt anlegen, benötigen Sie eine Seriennummer. Sie können diese bei jeder Anlage eines Arztes eingeben.

Beim Datenimport ist es jedoch angenehmer, wenn man eine Menge von Seriennummern einlesen kann und von diesen einer automatisch verwendet wird, sobald ein Arzt angelegt werden muss.

Siehe hierzu auch den Punkt Extras > Seriennummern verwalten

Drucken

nach oben

Auf vielen Seiten können Sie den Inhalt ausdrucken, wobei der Ausdruck über den Internet-Browser vorgenommen wird. Hier können Sie für die einzelnen Druckfunktionen einstellen, wie viele Zeilen auf einer Seite gedruckt werden.

Sonstiges

nach oben

Hier können Sie verschiedene Einstellungen vornehmen.

Wasserzeichen
Wenn Sie statt des grünen Männchens ein anders Bild anzeigen möchten, müssen Sie die Grafik in das Programmverzeichnis unter dem Namen 'Watermark.png' kopieren. Die empfohlene Größe ist 128 x 128 Pixel.
