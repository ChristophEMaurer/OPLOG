Lizenz installieren Assistent

Wenn Sie eine Lizenz erworben haben, bekommen Sie diese per EMail als Datei zugeschickt. Diese Datei speichern Sie auf Ihrem Computer, und rufen dann diesen Assistenten auf. Sie müssen nun die auf Ihrem Computer gespeicherte Datei auswählen und bestätigen.

Beim nächsten Start von OP-LOG müssen Sie keine Seriennummern eingeben, solange die Lizenzdatei gültig ist.
