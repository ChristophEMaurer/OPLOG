Datenbank komprimieren

Dieser Menüpunkt steht nur zur Verfügung, wenn Sie mit einer Microsoft Access Datenbank arbeiten.
Eine Datenbank wächst, je mehr Daten eingefügt werden. Wenn Daten gelöscht werden, wird die Datenbank allerdings nicht kleiner.
Hiermit haben Sie die Möglichkeit, die Datenbank einmal auf die notwendige Größe zu verkleinern.
Dieses sollte nur vom Systemadministrator durchgeführt werden und nicht ohne vorher eine Sicherung der Datenbankdatei durchgeführt zu haben.
