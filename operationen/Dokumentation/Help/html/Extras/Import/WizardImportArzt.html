Arzt importieren Assistent

Siehe auch Extras > Datenexport > Arzt exportieren Assistent.

Mit diesem Assistenten können Sie ihre eigenen Daten die Sie im Krankenhaus exportiert haben, in ein anderes System importieren. Dadurch werden im Wesentlichen alle Ihre ausgeführten Prozeduren in das System eingefügt.

Hinweis

Sie können exportierte Daten nur innerhalb derselben Programm-Version importieren, anderfalls erhalten Sie eine solche Fehlermeldung:

Lesen Sie hierzu Chirurgendaten exportieren und importieren.
