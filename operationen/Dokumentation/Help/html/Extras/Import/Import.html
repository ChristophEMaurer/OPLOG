Datenimport

Hier können Sie verschiedenen Daten des Systems importieren. Zu den Daten "Richtlinien", "Zuordnungen von OPS-Kodes zu Richtlinien" und "Arzt" gibt es jeweils einen Export und einen Import.

Für den automatischen Datenimport werden die Einstellungen im Fenster "Extras > Optionen > Automatischer Prozeduren-Import" festgelegt.

Für den manuellen Datenimport werden die Einstellungen direkt im Fenster "Ausgeführte Prozeduren importieren" festgelegt.
