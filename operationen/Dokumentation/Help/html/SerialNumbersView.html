Seriennummern verwalten

In diesem Fenster können Sie die von Ihnen erworbenen Seriennummern verwalten. Kopieren Sie einfach Ihre Seriennummern in das Textfeld unten und speichern Sie diese. Wenn Sie nun einen neuen Arzt anlegen, können Sie unter Extras > Optionen > Seriennummern einstellen, dass automatisch eine freie Seriennummer verwendet wird.

Wenn Sie einen Arzt löschen, wird dessen Seriennummer wieder in den Pool der freien Seriennummern aufgenommen.
