Automatischer Update

Wenn eine neuere Version des Programms im Internet zur Verfügung steht, kann diese automatisch heruntergeladen werden. Sie sehen dann als Fortschritt dieses Fenster. Vorher erscheint die folgende Rückfrage:

Sie können die Überprüfung, ob es eine neue Version im Internet gibt, manuell unter Extras > Auf neue Programmversion überprüfen anstoßen und unter Extras > Optionen > Programm Update einstellen, ob beim Programmstart diese Überprüfung automatisch vorgenommen werden soll.
